How to Use

Generating a QR Code
  1. 1Go to the Generator tab
  2. 2Enter your text, URL, or any data
  3. 3Customize color, background, and size
  4. 4Add an optional description (appears in download)
  5. 5Click Generate QR Code
  6. 6Download as PNG or JPG
Scanning / Decoding
  1. 1Go to the Scanner tab
  2. 2Upload or drag-and-drop a QR code image
  3. 3Click Decode QR Code
  4. 4The decoded data appears in the result box
  5. 5Non-QR images show an error message
